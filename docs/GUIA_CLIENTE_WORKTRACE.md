# Guia de Uso de WorkTrace

Fecha: 8 de abril de 2026

## 1. Para que sirve este sistema

WorkTrace ayuda a organizar el trabajo diario en restaurantes.

Con este sistema se puede:
- ver turnos
- iniciar y finalizar labores
- tomar fotos de evidencia
- revisar supervisiones
- crear empleados
- crear restaurantes
- programar turnos
- sacar informes

La idea es que todo sea claro y facil de usar.

## 2. Antes de comenzar

Antes de usar el sistema, tenga en cuenta lo siguiente:

1. Use un celular con internet.
2. Mantenga la bateria con buena carga.
3. Cuando el sistema pida permisos de camara o ubicacion, permita el acceso.
4. Si una pantalla tarda un poco, espere unos segundos y no pulse muchas veces el mismo boton.

## 3. Ingreso al sistema

Para entrar:

1. Escriba su correo y su contraseña.
2. Si el sistema pide un codigo, escriba el codigo de verificacion.
3. Si entra por primera vez, el sistema puede pedirle cambiar su contraseña.
4. Cuando termine de cargar, lo llevara a su pantalla principal.

## 4. Si usted es empleado

En la pantalla principal podra ver:
- el restaurante
- el horario
- la fecha
- el boton para iniciar o continuar el turno

### Iniciar turno

1. Revise que el restaurante y el horario sean correctos.
2. Pulse el boton para iniciar turno.
3. Permita la ubicacion si el celular la pide.
4. Tome las fotos solicitadas.
5. Cuando termine, el sistema dejara continuar.

### Durante el turno

Durante el turno el empleado puede:
- ver su informacion
- hacer tareas especiales, si le dejaron alguna
- tomar las fotos que correspondan

### Tarea especial

Si aparece una tarea especial:

1. Lea con calma lo que se le pide.
2. Haga la tarea.
3. Si el sistema pide foto, tome la evidencia.
4. Marque la tarea como realizada cuando termine.

### Finalizar turno

1. Pulse finalizar turno.
2. Tome las fotos finales.
3. Si termina antes de tiempo, escriba el motivo.
4. Confirme para cerrar el turno.

## 5. Si usted es supervisora

La supervisora puede hacer varias tareas desde su panel.

### Crear restaurante

1. Pulse Nuevo Restaurante.
2. Escriba el nombre.
3. Busque la direccion o use su ubicacion actual.
4. Revise el punto en el mapa.
5. Guarde.

### Crear empleado

1. Pulse Nuevo Empleado.
2. Escriba nombre, correo y telefono.
3. Guarde.

El empleado queda activo para poder programarse.

### Programar turnos

Hay tres formas:

1. Un solo dia
2. Varios empleados
3. Varios dias

Use la opcion que mejor le sirva.

Despues:

1. Elija empleado o empleados.
2. Elija restaurante.
3. Ponga fecha y horario.
4. Si hace falta, agregue una tarea especial.
5. Guarde la programacion.

### Supervisar en sitio

1. Entre a Supervisar.
2. Elija el restaurante.
3. Elija el turno a supervisar.
4. Elija el area.
5. Tome las fotos.
6. Escriba observaciones si hace falta.
7. Guarde la supervision.

### Informes

En informes puede:
- ver un informe en pantalla
- descargar PDF
- descargar Excel

Si el informe es de un solo dia, pueden salir las fotos de antes y despues.

## 6. Si usted es super admin

El super admin puede:
- ver el resumen general
- revisar restaurantes
- revisar empleados
- revisar supervisoras
- monitorear supervisiones del dia
- sacar informes

### Monitoreo de supervisoras

En esta parte puede:

1. ver cuantas supervisiones hubo hoy
2. ver que supervisora hizo cada visita
3. ver en que restaurante fue
4. ver la hora
5. ver observaciones y evidencias

Tambien puede filtrar por una supervisora en particular.

## 7. Recomendaciones sencillas

Para usar mejor el sistema:

1. Revise bien antes de guardar.
2. Tome fotos claras.
3. No cierre la pagina mientras este cargando.
4. Si algo no aparece, recargue una sola vez.
5. Si el problema sigue, contacte a la persona encargada.

## 8. Consejos para personas mayores

1. Use el celular en posicion vertical.
2. Toque un boton a la vez.
3. Espere a que termine cada pantalla.
4. Si el sistema pide camara o ubicacion, pulse Permitir.
5. Si no entiende una opcion, vuelva al inicio y entre de nuevo con calma.

## 9. Resumen rapido

Empleado:
- ve su turno
- toma fotos
- hace tareas
- finaliza turno

Supervisora:
- crea restaurantes
- crea empleados
- programa turnos
- hace supervisiones
- genera informes

Super admin:
- revisa todo el sistema
- monitorea supervisoras
- consulta resultados generales

## 10. Mensaje final

La idea de WorkTrace es ayudar, no complicar.

Si se usa paso a paso, el sistema permite llevar el control del trabajo diario de forma clara y ordenada.
